# 🍬 Candy Crush Game

A Candy Crush-inspired puzzle game built using **HTML, CSS, and JavaScript**. The game features drag-and-drop gameplay, match detection, score tracking, and dynamic candy generation for an engaging gaming experience.

## 🚀 Features

- 🎮 Drag-and-drop candy swapping
- 🍭 Match 3 or more candies
- ⭐ Real-time score tracking
- 🔄 Automatic candy refill
- 🎨 Responsive and interactive UI
- ⚡ Smooth gameplay using JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/candy-crush-game.git
   ```

2. Open the project folder.

3. Open `index.html` in your browser.

No additional setup or dependencies are required.

## 🎯 Gameplay

- Drag one candy onto an adjacent candy.
- Match **3 or more** candies of the same color.
- Earn points for every successful match.
- New candies automatically fill empty spaces.
- Continue matching to achieve the highest score.
